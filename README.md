# dotfiles

## Installation

### Requirements
* [neovim](https://neovim.io)
* [tmux](https://github.com/tmux/tmux)
* [alacritty](https://alacritty.org)
* [GNU stow](https://www.gnu.org/software/stow/)

```
# Clone dotfiles to home directory
# Install (symlink) dotfiles
stow zshell
stow tmux
stow neovim
```
